# 《骑马与砍杀2：霸主》中文百科编纂体例

## 编纂目标

本项目不是把官方英文文本逐句翻译成中文，也不是按字数扩写。目标是以官方英文文本为资料源，写成一套可阅读、可审校、可合并的中文百科/设定集。

每一卷都必须满足：

- 信息来自已提取的官方英文文本、原版模块数据或本仓库可追溯材料。
- 正文以原创中文归纳为主，不整段复制英文原文。
- 每章有明确主题，避免不同章节重复同一段论述。
- 重要判断尽量保留来源键、地名或人物名，方便回查源文本。
- 先保证内容质量，再考虑字数规模。

## 文件结构

- `volumes/`：分卷 Markdown 源稿。
- `docx/`：分卷 Word 和最终合并 Word。
- `notes/`：证据摘录、术语表、交叉引用表、待核查事项。

## 卷册规划

1. 卷一：卡拉迪亚总史与帝国崩裂。
2. 卷二：六大文化、王国制度与政治秩序。
3. 卷三：城市、村庄、城堡与地理志。
4. 卷四：家族、领主、人物与继承关系。
5. 卷五：任务、对话、社会生活与民间秩序。
6. 卷六：军事、兵种、战争、经济与物产。
7. 卷七：战帆 DLC、海洋、港口、舰队与贸易。
8. 卷八：年表、术语表、地名索引、人物索引、交叉引用。

## 写作原则

- 不用模板循环补字。
- 不把同一个结论在不同章节中原样重复。
- 一段只解决一个问题：历史、制度、地理、军事、经济或社会生活。
- 章节之间可以交叉引用，但不能复制段落。
- 每卷末尾保留“本卷待补证据”和“下一卷承接点”。
